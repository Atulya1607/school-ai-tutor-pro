# AI School Tutor 🤖📚

A production-ready generative AI web application designed to provide interactive tutoring and step-by-step problem-solving for students in Classes 8th to 12th.

## ✨ Features
- **Topic Explanations:** Breaks down complex subjects (Physics, Math, Chemistry, etc.) into simple language.
- **Instant Query Solver:** Solves academic questions with structured, step-by-step logic.
- **Class-Specific Tuning:** Dynamically adapts its explanation style based on the user's selected grade (8th - 12th).

## 🛠️ Tech Stack
- **Language:** Python 3
- **Framework:** Streamlit (Frontend & UI)
- **AI Engine:** Google Gemini API (LLM Integration)
- **Version Control:** Git & GitHub

## 📦 Installation & Setup
To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the application:
   ```bash
   streamlit run app.py
   ```
